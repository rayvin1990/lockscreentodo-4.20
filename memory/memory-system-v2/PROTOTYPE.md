## 📱 原型总览

### 用户旅程地图

```
用户启动                    选择功能                    查看结果                    后续操作
   │                           │                           │                           │
   ▼                           ▼                           ▼                           ▼
┌─────┐                    ┌─────┐                    ┌─────┐                    ┌─────┐
│ python│ ───────────────► │命令参数│ ───────────────► │结果展示│ ───────────────► │保存/ │
│main.py│                    │解析执行│                    │格式化  │                    │导出  │
└─────┘                    └─────┘                    └─────┘                    └─────┘
```

## 📐 交互流程

### 搜索流程

```
用户输入搜索命令
       │
       ▼
┌─────────────┐
│ 解析参数    │
│ - 关键词    │
│ - 天数范围  │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 遍历记忆文件│
│ (按日期倒序)│
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 关键词匹配  │
│ (不区分大小写)│
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 提取上下文  │
│ (前后 50 字)  │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 格式化输出  │
│ (带高亮)    │
└──────┬──────┘
       │
       ▼
  显示结果
```
