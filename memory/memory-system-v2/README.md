# 记忆系统 2.0 - Memory System V2

## 🎯 项目目标

解决当前记忆系统的痛点：
- ❌ 记忆文件分散，难以查找
- ❌ 没有自动提炼机制，重要信息容易丢失
- ❌ 提醒系统简陋，容易错过任务
- ❌ 缺乏搜索功能

## ✨ 核心功能

### 1. 自动记忆提炼
- 每天结束时自动读取当日 `memory/YYYY-MM-DD.md`
- 提取重要事件、决定、待办
- 自动追加到 `MEMORY.md`（长期记忆）

### 2. 智能提醒系统
- 基于任务优先级 + 截止时间
- 支持多种提醒方式（飞书消息、心跳检查）
- 静默时段自动跳过

### 3. 记忆搜索
- 关键词搜索所有记忆文件
- 按日期范围筛选
- 按标签/分类过滤

### 4. 任务管理
- 任务创建、更新、完成
- 优先级管理
- 过期任务告警
