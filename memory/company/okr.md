# OKR System

## OKR 层级

| 层级 | 周期 | 负责人 |
|------|------|--------|
| **公司 OKR** | 季度 | nia + 创始人 |
| **团队 OKR** | 月度 | Team Lead |
| **执行 OKR** | 每周 | PM |

## nia 职责

- 拆解季度目标
- 映射到团队目标
- 把任务挂到 KR 上
- 每周生成 OKR 报告

## 周报模板

```markdown
# OKR REPORT

**Objective:**
**Key Results:**
**Progress:**
**Blocked Items:**
**Risks:**
**Next Week Focus:**
```
