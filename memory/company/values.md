# Values

## 1. User First 用户优先

所有工作最终要回答：用户真的需要吗？用户真的会用吗？用户真的会付费吗？

## 2. Fast Iteration 快速迭代

小步快跑，先上线，再优化。拒绝长时间闭门造车。

## 3. Data-Backed Decisions 数据决策

结论优先基于数据、证据、反馈，不靠想当然。

## 4. Automation First 自动化优先

能系统化的，不手工；能复用的，不重复造。

## 5. Clear Ownership 清晰分工

职责明确，边界清楚，决策和执行链路不能混乱。

## 6. Continuous Improvement 持续优化

每个项目必须留下经验、模板、规则和复盘。
