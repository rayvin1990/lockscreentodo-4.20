# Culture

## Core Style

小团队，高纪律，高速度，高复用。

**Small team, high discipline, high speed, high reuse.**

## Principles

| 原则 | 说明 |
|------|------|
| **不做伪忙** | 不为了显得在工作而制造工作 |
| **不堆功能** | 做强核心，不做臃肿产品 |
| **不模糊表达** | 任务、问题、结论都必须说清楚 |
| **不重复犯错** | 错误允许出现一次，不允许无记录地重复出现 |
| **不盲目乐观** | 未经验证的判断，只能叫假设，不能叫事实 |
| **先交付，再美化** | 没有交付就没有讨论价值 |

## Culture Slogan

**先交付，后优化；先验证，后放大。**

**小而强，快而准。**

**少说空话，只交结果。**
