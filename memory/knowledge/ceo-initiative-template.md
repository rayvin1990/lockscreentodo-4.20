# CEO Initiative Template

## 创始人指令模板

**以后创始人按这个格式发指令：**

```
CEO INITIATIVE

Project:
Goal:
Project Type:
Constraints:
Expected Deadline:
Priority:
Notes:
```

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| **Project** | 项目名称 | AI Landing Page MVP |
| **Goal** | 项目目标 | 3 天内完成一个可上线的预购页 |
| **Project Type** | 项目类型 | MVP / 功能迭代 / 技术验证 |
| **Constraints** | 约束条件 | 核心功能不超过 3 个 |
| **Expected Deadline** | 期望截止时间 | 72 小时 |
| **Priority** | 优先级 | P0 / P1 / P2 |
| **Notes** | 备注 | 需要支付预留、基础埋点 |

## 示例

```
CEO INITIATIVE

Project: AI Landing Page MVP
Goal: 3 天内完成一个可上线的预购页，并为搜索增长预留结构
Project Type: MVP
Constraints: 核心功能不超过 3 个；首页、功能页、1 个对比页必须具备
Expected Deadline: 72 小时
Priority: P0
Notes: 需要支付预留、基础埋点、Search Console 接入准备
```

## nia 收到后的处理流程

1. 判断项目类型
2. 检查优先级
3. 任命 PM（如需要）
4. 启动相应责任槽位
5. 生成任务分配单
