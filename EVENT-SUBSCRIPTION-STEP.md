# 飞书事件订阅配置 - 具体操作步骤

## 📍 你现在在这里

你已经打开了页面：
```
https://open.feishu.cn/app/cli_a9f5af0e51f89cc4/event?tab=event
```

这个URL说明你在应用的**事件页面**，`tab=event` 表示你已经在事件选项卡中了。

---

## 🎯 现在需要做什么

### 步骤1：查找事件订阅选项

在你打开的页面中，寻找以下内容：

#### 选项A：直接看到事件订阅配置
页面上可能有：
- **"事件订阅"** 区域
- **"请求地址"** 输入框
- **"订阅事件"** 列表

#### 选项B：需要点击按钮
页面上可能有以下按钮：
- **"添加事件"** / **"Add Event"**
- **"配置事件"** / **"Configure Events"**
- **"订阅管理"** / **"Subscription Management"**

#### 选项C：有多个子标签
页面顶部可能有多个标签：
- **"事件订阅"** (Event Subscription) ← 点这个
- **"事件日志"** (Event Log)
- **"推送配置"** (Push Config)

---

### 步骤2：配置请求地址

找到输入框后，填写：

```
https://halolike-kensley-subventricous.ngrok-free.dev/webhook/feishu
```

**注意：**
- ✅ 必须是 https 开头
- ✅ 必须包含 `/webhook/feishu` 后缀
- ✅ ngrok URL 必须与你的 ngrok 窗口显示的一致

---

### 步骤3：订阅事件

这是**最重要**的一步！

在页面中找到事件订阅列表，添加以下事件：

#### 必需事件：
- ✅ **`im.message.receive_v1`**
  - 这个事件让机器人接收群聊消息
  - 作用：当有人在群里 @机器人 时，飞书会通知你的服务

#### 如何添加：
1. 找到 **"添加事件"** / **"+"** 按钮
2. 点击后，搜索或浏览事件列表
3. 找到并勾选 `im.message.receive_v1`
4. 点击 **"保存"** 或 **"添加"**

---

### 步骤4：验证配置

配置完成后：
1. 点击 **"验证"** / **"保存"** / **"应用"** 按钮
2. 等待验证成功提示（通常显示绿色勾选）

---

### 步骤5：发布机器人

回到应用详情页：
1. 找到 **"发布"** 按钮
2. 选择 **"企业内发布"**
3. 点击发布

---
