# Price to Hours v1.8.0 更新说明

## 主要改进

### 1. 增强的价格识别引擎
- **支持更多货币符号**: $ £ € ¥ ₽ ₹ ₩ ₺ ₴ ₦ ₿ 等
- **支持货币代码**: CNY, USD, EUR, GBP, JPY, RMB, HKD, TWD, KRW, INR, AUD, CAD, CHF, SEK, NOK, DKK, NZD, SGD, THB, MYR, PHP, IDR, VND
- **更灵活的价格格式**: 支持空格、不间断空格等多种格式

### 2. 全新的提示框 UI

#### 视觉效果
- **深色渐变背景**: 高级感的深蓝渐变设计
- **金色价格显示**: 使用渐变金色突出价格
- **毛玻璃效果**: backdrop-filter 模糊背景
- **精致边框**: 半透明白边框 + 阴影效果

#### 内容布局
```
┌─────────────────────────┐
│      $299.00           │ ← 金色大字体
├─────────────────────────┤
│  ≈ 2.0 小时             │ ← 白色主信息
│  一集电视剧的时间        │ ← 绿色情绪文案
│  (120 分钟)            │ ← 灰色辅助信息
└─────────────────────────┘
```

### 3. 情绪价值文案

根据工时长度自动显示不同的趣味文案：

| 工时范围 | 主显示 | 情绪文案 |
|---------|--------|----------|
| < 6 分钟 | ≈ 5 分钟 | 转眼就赚到啦！ |
| < 1 小时 | ≈ 45 分钟 | 喝杯咖啡的时间 |
| 1-2 小时 | ≈ 1.5 小时 | 一集电视剧的时间 |
| 2-4 小时 | ≈ 3.0 小时 | 半个上午就搞定 |
| 4-8 小时 | ≈ 6.5 小时 | 一上午/一下午 |
| 1-2 天 | ≈ 1.5 工作日 | 一天不到/一天多一点点 |
| 2 天 -1 周 | ≈ 3.5 工作日 | 差不多一周啦 |
| 1 周 -1 月 | ≈ 2.5 周 | 两周不到/半个多月 |
| 1 月 -1 年 | ≈ 6.0 月 | 小半年/大半年啦 |
| > 1 年 | ≈ 1.5 年 | 这需要攒很久哦 |

### 4. 三级信息显示
- **主要信息**: 最直观的时间单位（小时/工作日/周/月/年）
- **次要信息**: 情绪化文案，绿色斜体
- **辅助信息**: 其他单位换算，灰色小字

## 技术改进

### 计算器函数
```javascript
// 新增详细计算函数
calculateWorkTimeDetail(price, amount, incomeType, hoursPerDay)
// 返回：
{
  hours: 2.5,

  days: 0.31,
  weeks: 0.06,
  months: 0.01,
  primary: "≈ 2.5 小时",
  secondary: "半个上午就搞定",
  tertiary: "(150 分钟)"

```

### CSS 变量
- 暗色模式：`#1a1a2e` → `#16213e` 渐变
- 金色价格：`#ffd700` → `#ffb347` 渐变
- 绿色文案：`#4ade80`
- 辅助文字：`#888`

## 使用示例

### 悬停效果
1. 鼠标悬停在价格上（如 `$299`）
2. 150ms 后显示精美提示框
3. 移动鼠标，提示框跟随移动
4. 鼠标移开，提示框立即消失
