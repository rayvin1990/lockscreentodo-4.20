# iOS Shortcuts 一键设置壁纸指南

## 🎯 功能说明

通过 iOS Shortcuts，用户扫描二维码后可以：
1. 自动下载壁纸
2. 自动设置为锁屏
3. 只需确认一次，全程自动化

---

## 📱 创建 iOS Shortcuts

### 步骤 1：打开 Shortcuts App
在 iPhone 上找到并打开 "快捷指令" App

### 步骤 2：创建新快捷方式
1. 点击右上角 "+" 号
2. 点击 "添加操作"

### 步骤 3：添加操作

#### 操作 1：获取 URL 内容
1. 搜索 "URL 的内容" 或 "Get Contents of URL"
2. 配置：
   - URL：选择 "快捷方式输入" 或 "Shortcut Input"
   - 这会自动接收二维码传递的 URL

#### 操作 2：设置为壁纸
1. 搜索 "设定壁纸" 或 "Set Wallpaper"
2. 配置：
   - 选中 "显示预览"（可选）
   - 选中 "设为锁屏"（Lock Screen）
   - 取消"主屏幕"（Home Screen）

### 步骤 4：配置快捷方式
1. 点击右上角图标，进入设置
2. 重命名为：设置锁屏壁纸
3. 添加到主屏幕（可选）
4. 接受输入类型：选择 "URL"

### 步骤 5：共享快捷方式
1. 点击右上角分享按钮
2. 选择 "拷帕快捷方式"
3. 打开 iCloud 链接：https://www.icloud.com/shortcuts/
4. 点击 "+" 号，粘贴你的快捷方式
5. 复制分享链接，格式如：`https://www.icloud.com/shortcuts/xxx`

---

## 🔗 快捷方式 URL 格式

生成的二维码 URL 应该是：
```
https://www.icloud.com/shortcuts/你的快捷方式ID?wallpaper=https://你的网站.com/api/share/图片ID
```

---

## ⚠️ 重要提示

1. **快捷方式参数**：快捷方式需要接收一个名为 `wallpaper` 的 URL 参数
2. **图片 URL**：你的网站需要提供可公开访问的图片 URL
3. **HTTPS 要求**：图片 URL 必须是 HTTPS
4. **尺寸限制**：建议图片不超过 5MB

## 🎨 用户使用流程

### 电脑上生成：
```
1. 用户在网站上生成壁纸
2. 看到二维码（包含快捷方式链接 + 壁纸 URL）
3. 用 iPhone 相机扫码
4. 点击"打开快捷方式"
5. 快捷方式自动下载并设置壁纸
6. 确认完成 ✅
```

### 手机上生成：
```
1. 用户在 iPhone 上生成壁纸
2. 点击"一键设置"按钮
3. 直接打开快捷方式
4. 自动下载并设置壁纸
5. 确认完成 ✅
```

## 🛠️ 快捷方式高级配置（可选）

### 添加错误处理
1. 在快捷方式开始添加 "如果" 操作
2. 条件：输入 不为空
3. 否则：显示通知 "请先生成壁纸"

### 添加完成提示
1. 在快捷方式末尾添加 "显示通知" 或 "显示提醒"
2. 内容：壁纸已设置完成 ✅

### 添加保存到相册
如果用户想要同时保存：
1. 在 "设为壁纸" 之前添加 "保存到相册" 操作
2. 这样图片会同时保存到相册

## 📊 测试你的快捷方式

1. 创建一个测试图片 URL
2. 在 Safari 中访问：`shortcuts://run-shortcut?text=[参数名]&input=https://测试图片.jpg`
3. 验证是否能正确下载和设置

## 🔧 故障排除

### 问题 1：快捷方式不执行
- 检查快捷方式是否接受 URL 输入
- 检查 URL 参数名是否为 `wallpaper`

### 问题 2：图片无法下载
- 确认图片 URL 可以在浏览器中访问
- 确认使用 HTTPS
- 检查 CORS 配置

### 问题 3：无法设置壁纸
- 确认快捷方式有相册访问权限
- 检查 iOS 版本（需要 iOS 13+）
