## 📱 iOS Shortcuts 配置（关键步骤）

### 选项 A：使用现有的快捷方式（测试）
如果你已经有 iOS Shortcuts：
1. 在 `.env.local` 中配置：
   ```bash
   NEXT_PUBLIC_IOS_SHORTCUT_URL='https://www.icloud.com/shortcuts/你的快捷方式ID'
   ```

### 选项 B：创建新的快捷方式（推荐）
1. 打开 iPhone 上的"快捷指令"App
2. 按照 `IOS_SHORTCUTS_GUIDE.md` 中的详细步骤创建
3. 配置快捷方式接收参数：`wallpaper`
4. 添加两个操作：
   - "URL 的内容"（获取壁纸）
   - "设定壁纸"（设为锁屏）
5. 分享快捷方式，获取 URL

### 选项 C：暂时不配置（测试二维码）
如果不配置 iOS Shortcuts URL：
- 二维码仍然会显示
- 扫码会直接打开图片
- 用户可以手动保存并设置

## 🔧 故障排除

### 问题1：看不到二维码
**解决**：
- 确认是否生成了壁纸
- 检查浏览器控制台错误（F12 → Console）
- 确认 `/api/share` 路由是否正常

### 问题2：iOS Shortcuts 不执行
**解决**：
- 确认快捷方式已配置
- 确认快捷方式接受 URL 参数
- 确认快捷方式有相册权限

### 问题3：二维码扫描后无法打开图片
**解决**：
- 确认图片 URL 是 HTTPS
- 确认 URL 可以在浏览器中访问
- 检查 CORS 配置
