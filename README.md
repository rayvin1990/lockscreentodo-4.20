# Price to Hours

**See prices as hours of work. Shop smarter, spend wiser.**

---

## ⚡ 3-Second Setup

1. **Install** the extension
2. **Hover** over any price
3. **See** how many hours of work it costs

**That's it.**

---

## 🎯 Default Settings (Ready to Use)

- **Income:** $3,000 / Biweekly
- **Work Hours:** 8 hours/day
- **Status:** Enabled

No setup required. Works out of the box.

---

## ⚙️ Want to Customize?

1. Click the extension icon ⏱️
2. Enter your income (e.g., `3000`)
3. Select pay period (Hourly/Daily/Weekly/Biweekly/Monthly/Yearly)
4. Set hours per day (default: 8)
5. Click **Save Settings**

Done in 30 seconds.

---
