# Price to Hours - Icon Placeholder

This directory needs icon files for the extension.

## Required Icons

- `icons/icon16.png` - 16x16 pixels
- `icons/icon32.png` - 32x32 pixels
- `icons/icon128.png` - 192x192 pixels

## Icon Design Suggestion

- **Background**: Black (#111)
- **Text**: White "⏱️" emoji
- **Style**: Simple, clean

## Quick Generation

Visit https://favicon.io/ and:
1. Upload or select ⏱️ emoji
2. Set background color to black
3. Download all sizes (16, 48, 128)
4. Rename files as shown above

## Alternative

Use this SVG code:

```svg
<svg width="128" height="128" xmlns="http://www.w3.org/2000/svg">
  <rect width="128" height="128" fill="#111" rx="16"/>
  <text x="64" y="80" font-size="64" text-anchor="middle" fill="white">⏱️</text>
</svg>
```

Convert to PNG using any SVG to PNG tool.
